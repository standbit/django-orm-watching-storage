# django-orm-watching-storage
Репозиторий с сайтом для урока «Пишем пульт охранника банка» курса dvmn.org

Программа позволяет отслеживать, кто находится в хранилище банка в данный момент времени. Выводит всех владельцев активных пропусков. Показывает, кто, когда и как долго находился в харнилище. Также присваивает флаг `True` подозрительным охранникам.
